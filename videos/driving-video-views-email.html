Generar reproducciones de videos a través del correo electrónico

{{ page.description }}

La función Publicar en Brightcove Campaign™ se puede utilizar para generar fácilmente el código de inserción y las imágenes necesarias para distribuir un video mediante una página de destino y un correo electrónico. Por lo general, el código de inserción del video se pega en una página de destino y luego se envía un enlace a la página de destino por correo electrónico. El proceso descrito en este tema se adapta mejor cuando se utiliza una plataforma de automatización de marketing (MAP). no siendo utilizado. Si está utilizando un MAP, Brightcove recomienda utilizar el proceso de publicación.

Creando una página de aterrizaje

Una página de destino es una página web independiente que se puede utilizar como parte de una campaña de marketing para promocionar un producto o servicio. Agregar videos a las páginas de destino es una excelente manera de aumentar la participación de los espectadores y convertirlos en compradores. Hay varios proveedores que ofrecen la posibilidad de crear páginas de destino. Las páginas de destino también se pueden crear utilizando un software de plataforma de automatización de marketing o manualmente, utilizando HTML personalizado si tiene recursos para desarrolladores. Una vez que haya creado una página de destino, Brightcove Campaign se puede utilizar para insertar un video en la página de destino.

Generando el código de inserción

Se requiere el código de inserción para que un video se reproduzca en una página de destino. Para copiar el código de inserción de un vídeo, sigue estos pasos:

  1. Abre el Videos página.
  2. Seleccione un video haciendo clic en la casilla de verificación junto al video que desea usar.
  3. Haz clic en Publicar.
  4. Copia el Código de inserción y péguelo en su página de destino donde desea que aparezca el video.
  5. Guarde los cambios en su página de destino.
  6. Publique la página de destino y asegúrese de tener una URL en la que se pueda ver la página de destino.

Copia de la imagen de póster

En la mayoría de las plataformas, no es’posible insertar vídeos directamente en el correo electrónico. Esta no es una limitación de Brightcove, casi todos los sistemas de correo electrónico modernos eliminarán el código de inserción del reproductor de vídeo por razones de seguridad. Sin embargo, podemos generar conciencia y tráfico de contenido creando un correo electrónico con texto de adelanto y una imagen de póster. La imagen del póster se verá como un reproductor de video y será un enlace en el que se puede hacer clic que redirige a la página de destino.

Para copiar la imagen del póster de un vídeo, sigue estos pasos:

  1. Haga clic en el Correo electrónico pestaña en la parte superior de la página.
  2. Haz clic en Copiar para copiar la imagen del póster
  3. Pega la imagen en la herramienta que enviará el correo electrónico.
  4. Cree un hipervínculo desde la imagen a la página de destino. El proceso para hacer esto variará según el programa de correo electrónico que se utilice.

Cuando se envía el correo electrónico, los lectores verán el video como parte del correo electrónico y, cuando hagan clic en la imagen, serán dirigidos a la página de destino donde pueden ver el video.
