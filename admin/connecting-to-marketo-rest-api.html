Conectarse a Marketo mediante las API REST

{{ page.description }}

Brightcove Campaign se puede configurar para integrarse con Marketo. Cuando se configura la integración, los datos de interacción de vídeo de los vídeos publicados con Brightcove Campaign se sincronizan con Marketo. Una vez en Marketo, estos datos se pueden utilizar para segmentación, lógica de campaña, informes, puntuación de clientes potenciales y personalización de comunicaciones.

Nota: Para obtener información sobre cómo configurar Brightcove Campaign para su uso con Marketo utilizando el ID de Munchkin, consulte Conexión a Marketo usando Munchkin.

Requisitos

Para utilizar Brightcove Campaign con la integración REST de Marketo, debe:

  • Cree un rol en Marketo que tenga acceso a todos los permisos de la API
  • Crear un servicio LaunchPoint

Creación de un usuario solo de API y servicio LaunchPoint

Antes de poder realizar una llamada a la API REST de Marketo, debe crear un nuevo rol de API con acceso a todas las API, crear un usuario solo de API y luego crear un servicio que identifique de forma única su aplicación cliente. Los pasos para completar estas tareas se describen en el Documentación de Marketo.

Configuración de la conexión

Una vez que se hayan completado los pasos de configuración en el lado de Marketo, siga estos pasos para configurar la conexión entre Brightcove Campaign y Marketo.

Establecimiento de la conexión entre Brightcove Campaign y Marketo

  1. Inicia sesión en Brightcove Campaign.
  2. En el encabezado de navegación, haz clic en el icono de engranaje () y luego Conexiones MAP. Aparecerá una lista de las conexiones actuales (si las hay).
  3. Haga clic en Agregar nueva conexión.
  4. Selecciona Marketo en la lista desplegable.
  5. Selecciona el Estándar (API REST) tipo de integración. Deberá pegar varios valores de su cuenta de Marketo. Tenga en cuenta que debe iniciar sesión como administrador de Marketo para completar los siguientes pasos.
  6. Introduce tu ID de cuenta de Marketo. Para obtener este valor, en Marketo haga clic en Administración > Munchkin.
  7. Entra tu URL del punto final de la API REST de Marketo. Para obtener este valor, en Marketo haga clic en Administración > Servicios web. Copia el Punto final en el API REST sección.
  8. Entra tu URL de identidad de la API REST de Marketo. Para obtener este valor, en Marketo haga clic en Administración > Servicios web.
  9. Ingresar ID de cliente de Marketo LaunchPoint. Para obtener este valor, en Marketo haga clic en Administración > LaunchPoint y luego haga clic en el Ver detalles enlace para el LaunchPoint que creó anteriormente como parte de esta configuración.
  10. Ingresar Secreto de cliente de Marketo LaunchPoint. Para obtener este valor, en Marketo haga clic en Administración > LaunchPoint y luego haga clic en el Ver detalles enlace para el LaunchPoint que creó anteriormente como parte de esta configuración.
  11. Haz clic en Guardar.

La conexión a Marketo ya está configurada y lista para usar. De forma predeterminada, el estado de la conexión estará habilitado. El estado de conexión determina si los datos de interacción de vídeo se envían a Marketo:

  • Habilitado: los datos se recopilan y se enviarán a Marketo
  • Deshabilitado: no se recopilan datos (la integración está deshabilitada)

Haz clic en Guardar para guardar los cambios. Para eliminar la integración, haz clic en Eliminar.
