Administración de la configuración de cuentas

{{ page.description }}

Para acceder a la campaña Brightcove™ configuración de la cuenta, haga clic en el icono de rueda dentada () en el encabezado de navegación.

Nota: Debe ser un administrador de cuenta para acceder al menú Administración.

El menú de configuración de la cuenta ofrece las siguientes opciones:

  • Información de cuenta
  • Administración de usuarios
  • Configuración de cuenta
  • Uso compartido de medios
  • Google analitico
  • Adobe Analytics
  • Conexiones MAP
  • Formularios de clientes potenciales
  • Estilo del jugador
  • Facturación

Información de cuenta

La opción Información de cuenta permite editar el nombre de la cuenta y la dirección postal. Para obtener información, consulte Administrar la información de la cuenta.

Administración de usuarios

La opción Gestión de usuarios le permite gestionar los usuarios autorizados para utilizar la aplicación Brightcove Campaign. Para obtener información, consulte Administrar usuarios.

Configuración de cuenta

La configuración de la cuenta le permite establecer la zona horaria de informes predeterminada para la cuenta y habilitar el campo Tipo de contenido de video. La zona horaria de informes se utiliza para calcular los límites diurnos de los datos de informes. Cambiar la configuración de zona horaria de informes solo afecta a los datos en el futuro y los cambios no se aplican de forma retroactiva. Si cambia esta configuración, es posible que vea un punto plano o un aumento en los datos de análisis al examinar los límites diurnos en los que se aplicó el cambio. Es posible que las actualizaciones de esta configuración no surtan efecto inmediatamente y los datos de análisis pueden seguir haciendo referencia a la configuración anterior hasta que el sistema procese los cambios.

Habilitando el Tipo de contenido de video En el campo, Brightcove puede compilar sus análisis con los de videos similares y proporcionar análisis más sólidos que incluyen la puntuación de referencia de video (VBS) de Brightcove y asesoramiento personalizado.

Uso compartido de medios

La opción Compartir contenido multimedia le permite cargar vídeos una vez y luego compartirlos en otras cuentas de Brightcove. Para obtener información, consulte Administrar la configuración de uso compartido de medios.

Google analitico

La opción de Google Analytics permite que los eventos de los jugadores y los datos de dimensiones personalizadas se envíen a una cuenta de Google Analytics para fines de informes. Para obtener información, consulte Configuración del seguimiento de eventos de Google Analytics.

Adobe Analytics

La opción Adobe Analytics permite que los datos de seguimiento de eventos se envíen a una cuenta de Adobe Analytics para fines de informes. Para obtener información, consulte Configuración del seguimiento de eventos de Adobe Analytics.

Conexiones MAP

La opción MAP Connections le permite configurar integraciones con plataformas de automatización de marketing como HubSpot, Eloqua y Marketo. Para obtener información, consulte Configuración de las conexiones de la plataforma de automatización de marketing.

Formularios de clientes potenciales

La opción Formularios de clientes potenciales le permite crear formularios de clientes potenciales que se pueden mostrar durante la reproducción de video. Para obtener información, consulte Creación de formularios de clientes potenciales.

Estilo de jugador

La opción Estilo del jugador le permite personalizar la apariencia del jugador, los colores y la superposición del jugador. Para obtener información, consulte Jugadores de estilo.

Facturación

La opción de facturación le permite ver la información de facturación de la cuenta. Para obtener información, consulte Ver información de facturación.
