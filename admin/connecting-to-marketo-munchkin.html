Conexión a Marketo usando Munchkin

{{ page.description }}

Brightcove Campaign se puede configurar para integrarse con Marketo. Cuando se configura la integración, los datos de interacción de vídeo de los vídeos publicados con Brightcove Campaign se sincronizan con Marketo. Una vez en Marketo, estos datos se pueden utilizar para segmentación, lógica de campaña, informes, puntuación de clientes potenciales y personalización de comunicaciones.

Nota: Para obtener información sobre cómo configurar Brightcove Campaign para su uso con Marketo utilizando las API REST, consulte Conectarse a Marketo mediante las API REST.

Algunas de las características clave de la integración de Marketo son:

  • Formato de datos - Los datos de visualización de videos se envían a Marketo como una vista de página o una visita virtual a la web. El evento de visita a la página web contiene parámetros de consulta que indican el nivel de participación del espectador. Los datos que se replican en Marketo son los siguientes:
    • Nombre del vídeo
    • ID de vídeo
    • ID de cuenta de Brightcove
    • URL de página de la página de referencia
    • ID de jugador de Brightcove
    • % observado
  • Transferencia de datos - Los datos de visualización se envían inmediatamente a Marketo; sin embargo, algunos informes en la plataforma de Marketo pueden tardar un tiempo en actualizarse.
Para utilizar Brightcove Campaign con Marketo Munchkin, debe tener una cuenta de Marketo con la API de Munchkin habilitada; necesitará su ID de cuenta de Munchkin (consulte Accediendo al Marketo Munchkin debajo). Los usuarios con la función de usuario de Munchkin deberían poder acceder a estos valores.

Accediendo al Marketo Munchkin

Al configurar la integración con Brightcove Campaign, se le pedirá su Marketo munchkin. Siga los pasos a continuación para acceder a este valor.

  1. Inicie sesión en su cuenta de Marketo.
  2. Hacer clic Administración en la navegación superior.
  3. En la barra de navegación de la izquierda, navegue hasta Administración > Integración > Munchkin.
  4. Copia el ID de cuenta de Munchkin y guárdelo en un lugar seguro. Será necesario al configurar la integración.

Configuración de la conexión

Siga estos pasos para configurar la conexión entre Brightcove Campaign y Marketo.

Establecimiento de la conexión entre Brightcove Campaign y Marketo

  1. Inicia sesión en Brightcove Campaign.
  2. En el encabezado de navegación, haz clic en el icono de engranaje () y luego Conexiones MAP. Aparecerá una lista de las conexiones actuales (si las hay).
  3. Haga clic en Agregar nueva conexión.
  4. Selecciona Marketo en la lista desplegable.
  5. Selecciona el Alto volumen (Munchkin) tipo de integración.
  6. Introduce tu ID de cuenta de Marketo. Este es el ID de cuenta de Munchkin copiado de Marketo.
  7. Haz clic en Guardar.

La conexión a Marketo ya está configurada y lista para usar. De forma predeterminada, el estado de la conexión estará habilitado. El estado de conexión determina si los datos de interacción de vídeo se envían a Marketo:

  • Habilitado: los datos se recopilan y se enviarán a Marketo
  • Deshabilitado: no se recopilan datos (la integración está deshabilitada)

Haz clic en Guardar para guardar los cambios. Para eliminar la integración, haz clic en Eliminar.
