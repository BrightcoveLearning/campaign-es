Configuración de las conexiones de la plataforma de automatización de marketing

{{ page.description }}

Campaña Brightcove™ se puede configurar para integrarse con plataformas de automatización de marketing (MAP) populares como HubSpot, Eloqua y Marketo. Cuando se configura la integración de MAP, los datos de participación de video de los videos publicados con Brightcove Campaign se sincronizan con la plataforma de automatización de marketing seleccionada.

Nota: Debe ser administrador de cuentas para acceder al menú de administración.
Nota: El proceso de conexión abrirá ventanas emergentes en su navegador. Debe deshabilitar cualquier bloqueador de ventanas emergentes del navegador antes de configurar la conexión.

Para configurar una integración con una plataforma de automatización de marketing, siga estos pasos:

  1. Inicia sesión en Brightcove Campaign.
  2. En el encabezado de navegación, haz clic en el icono de engranaje () y luego Conexiones MAP. Aparecerá una lista de las conexiones actuales (si las hay).
  3. Haga clic en Agregar nueva conexión.
  4. Seleccione el tipo de conexión. Consulte el tema correspondiente para obtener información sobre cómo conectarse a esa plataforma.

    • HubSpot (API REST o API del lado del cliente)
