Creación de formularios de clientes potenciales

{{ page.description }}

Los formularios de clientes potenciales se pueden utilizar para capturar información del espectador durante la reproducción de video. Al crear un formulario de cliente potencial, se le pedirá que pegue el HTML del formulario de cliente potencial desde una plataforma de automatización de marketing y luego configure la sincronización y visualización del formulario. Para obtener más información sobre cómo crear el formulario HTML de clientes potenciales, consulte el documento correspondiente a su plataforma de automatización de marketing:

  • Oracle Eloqua
  • Marketo
  • HubSpot
  • Salesforce

Crear un formulario de cliente potencial

Para crear un formulario de cliente potencial, siga estos pasos.

  1. Inicia sesión en Brightcove Campaign.
  2. En el encabezado de navegación, haz clic en el icono de engranaje () y luego Formularios de clientes potenciales. Aparecerá una lista de todos los formularios de clientes potenciales (si los hay).
  3. Hacer clic Crear formulario de cliente potencial.
  4. Pega tu Código de formulario de cliente potencial. Este es el formulario HTML generado a partir de su software de plataforma de automatización de marketing.
  5. Seleccione un Cerrar Confirmación tiempo de visualización. Esto determina cuánto tiempo se mostrará el mensaje de confirmación de cierre (si se ingresó).
  6. (Opcional) Introducir el Código de pantalla de confirmación. Este es el mensaje que se mostrará cuando se envíe el formulario de cliente potencial.
  7. Dale a la forma de plomo un descriptivo Nombre.
  8. Seleccione un Estado del formulario de cliente potencial.
    • Activo
    • Inactivo
  9. Seleccione el formulario Momento. El tiempo determina cuándo se mostrará el formulario de cliente potencial.
  10. Cheque Requerir envío para ver el video para solicitar que se envíe el formulario de cliente potencial para ver un video.
  11. Comprobación Muestre siempre el formulario de cliente potencial incluso si el cliente potencial ya es conocido siempre mostrará el formulario de cliente potencial incluso si ya se conoce el cliente potencial. Si no se marca, el formulario de cliente potencial no se muestra si se conoce el cliente potencial.
  12. (Opcional) Hacer clic Formulario de vista previa para abrir una vista previa del formulario en una nueva pestaña del navegador. El cuadro de diálogo de vista previa se puede utilizar para obtener una vista previa del formulario en dispositivos móviles, tabletas o de escritorio.
    Nota: Si la vista previa no aparece, asegúrese de que su navegador no haya bloqueado la aparición de la vista previa.
  13. Hacer clic Ahorrar para guardar el formulario de cliente potencial.

El formulario de cliente potencial se puede editar haciendo clic en el nombre del formulario de cliente potencial o marcando la casilla junto al formulario de cliente potencial y luego haciendo clic en Editar.

Ver datos del formulario de clientes potenciales

Los datos enviados en formularios de clientes potenciales se pueden ver en la plataforma de automatización de marketing adecuada. Los pasos para ver los datos se tratan en los temas del formulario de clientes potenciales para cada plataforma:

  • Oracle Eloqua
  • Marketo
  • HubSpot
  • Salesforce
