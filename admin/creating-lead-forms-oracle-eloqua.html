Creación de formularios de clientes potenciales en Oracle Eloqua

{{ page.description }}

Brightcove Campaign ofrece la posibilidad de crear formularios de clientes potenciales que le permiten capturar información del espectador durante la reproducción de vídeo. Los formularios le ofrecen la posibilidad de incluir los campos que necesite (visibles u ocultos) y también le ofrecen la posibilidad de aprovechar funciones como la creación de perfiles progresivos (si está disponible). Los datos de los clientes potenciales son enviados y procesados por Eloqua.

Creando un formulario en Eloqua

Eloqua brinda la capacidad de crear formularios que se pueden usar para capturar información del espectador. El HTML del formulario que se genera se utilizará al crear un formulario de clientes potenciales dentro de Brightcove Campaign. Los datos que se capturan se pueden utilizar para personalizar las comunicaciones futuras. Los pasos que se indican a continuación crearán un formulario sencillo para capturar el nombre, los apellidos y la dirección de correo electrónico. Para obtener detalles completos sobre Eloqua Forms, consulte la Documentación de Eloqua.

  1. Inicia sesión en tu cuenta de Eloqua.
  2. Navegar a Activos > Formularios y luego haga clic en Crear un formulario. Se abre el Selector de plantillas de formulario.
  3. Hacer clic Formulario en blanco y luego haga clic en Escoger.
  4. En la barra de navegación de la izquierda, expanda el Grupos de campo panel.
  5. En el Grupos de campo panel, haga clic en Información de contacto básica y arrástrelo a la plantilla de formulario.
  6. Confirme que los campos se hayan agregado al formulario.
  7. De forma predeterminada, los campos se agregan mediante una combinación de campos. Esto hará que aparezca un texto extraño en los campos cuando se use en Brightcove Campaign. Para eliminar la combinación de campos, seleccione el Primer nombre campo haciendo clic en él.
  8. En el panel de navegación de la izquierda, ubique el Propiedades generales sección y establezca el Rellenar previamente los datos del campo propiedad a Ninguno.
  9. Repita el proceso para Apellido y Dirección de correo electrónico campos.
  10. Haz clic en Guardar.
  11. Asigne un nombre al formulario y luego haga clic en Ahorrar.

Copiando el formulario HTML

Para utilizar el formulario de Brightcove Campaign, deberás copiar el HTML del formulario. Siga estos pasos para copiar el formulario HTML.

  1. Haga clic en Acciones > Ver HTML de formulario... Se mostrará el HTML del formulario.
  2. Copie todo el HTML en el portapapeles. Haz clic en Cerrar.
  3. Pega el HTML del formulario en el editor HTML de tu elección. Es posible que desee utilizar la función de formato de código de su editor para dar formato al código de modo que sea más fácil de leer.

Estilo de la forma

De forma predeterminada, cuando se muestra el formulario de cliente potencial, el formulario se mostrará sobre un fondo transparente sobre el reproductor. Querrá darle un estilo básico al formulario usando CSS. Las secciones siguientes proporcionan dos ejemplos de cómo se puede aplicar estilo al formulario. También se proporciona el CSS asociado. Es posible que deba adaptar el CSS para que se ajuste a sus colores estándar o para manejar el estilo de campos adicionales.

Creación de un formulario de clientes potenciales de Brightcove Campaign

Una vez creado y diseñado el formulario, el HTML del formulario se utilizará dentro de Brightcove Campaign al crear un formulario de clientes potenciales. Para obtener información sobre cómo crear un formulario de clientes potenciales, consulte Creación de formularios de clientes potenciales.

Validando los datos del formulario en Eloqua

Hay varias formas de validar que los datos del formulario se hayan pasado a Eloqua. Siga estos pasos para ver los datos del formulario en Eloqua.

  1. Inicia sesión en tu cuenta de Eloqua.
  2. Navegar a Activos > Formularios.
  3. En la barra de navegación de la izquierda, haga clic en el Todos los archivos enlace para mostrar todos los formularios.
  4. Haga doble clic en el nombre del formulario para abrirlo.
  5. Haga clic en Acciones > Ver datos de envío... Se mostrarán los datos del formulario.
